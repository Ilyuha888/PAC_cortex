# No test infrastructure for mocking VmClient and LLMClient

## Signal

tests/ has no conftest.py and no mock factories for VmClient or LLMClient. Every agent/runner test requires live SDK calls, making isolated unit testing impossible.

## Constraints

- Must not require live SDK/network for unit tests
- Mocks must faithfully reproduce SDK return shapes (dict[str,Any])

## Optimization Targets

- Minimal conftest.py with reusable VmClient and LLMClient fixtures
- Tests runnable with uv run pytest and no env vars

## Acceptance

conftest.py exists with mock_vm_client and mock_llm_client fixtures; at least one agent or runner test uses them and passes with no network calls

## Blast Radius

Blocks all agent.py and runner.py unit tests. Without mocks, test suite cannot grow beyond the existing safety module tests.

## Reversibility

high
