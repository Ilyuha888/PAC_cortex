# Agent fails 7/20 tasks due to 4 behavioral patterns: premature outcome classification, instruction scope miss, first-plausible-file bias, and inferred cleanup

## Signal

Run of 20 tasks scored 13/20 (65%). Seven failures cluster into four distinct patterns: (1) agent short-circuits to OUTCOME_NONE_UNSUPPORTED on surface keyword 'email' without exploring the workspace — the workspace has an outbox/ file system where 'send email' = write a file (t11, t14, t17); (2) same short-circuit produces wrong outcome: task expects NONE_CLARIFICATION because contact doesn't exist, but agent never looked (t12); (3) agent's plan covers only the most obvious clause of the task instruction and misses a secondary requirement (thread linking in t03); (4) agent stops at the first plausible entity file and doesn't cross-check related entities (t13: wrote reminders/rem_001.json, grader expected accounts/acct_001.json); (5) agent infers inbox cleanup (delete processed message) from convention, not instruction, causing unexpected FileDeleted (t19).

## Constraints

- Fixes must be behavioral invariants — general rules that improve reasoning, not task-specific lookup tables
- No new code paths or tools unless unavoidable — SYSTEM_PROMPT changes preferred
- Must not break the 13 passing tasks (especially the correct NONE_UNSUPPORTED calls on t04/t05/t06/t15)
- Total SYSTEM_PROMPT length must stay reasonable — no multi-page prompt

## Optimization Targets

- Eliminate premature outcome classification without exploring the workspace
- Ensure agent plan covers ALL clauses in task instruction before acting
- Prevent out-of-scope file deletions caused by inferred convention

## Observation Indicators

- API calls per task (don't balloon explore steps on already-clear tasks)
- False NONE_UNSUPPORTED rate (t04/t05/t06/t15 must stay correct)
- OUTCOME_OK rate on t11/t14/t17 (file-based email workflow tasks)

## Acceptance

">= 17/20 score on next run; t11/t14/t17 correctly resolve via outbox file write; t12 resolves to NONE_CLARIFICATION; t03 includes thread link; t13 hits accounts file; t19 doesn't delete inbox message

## Blast Radius
