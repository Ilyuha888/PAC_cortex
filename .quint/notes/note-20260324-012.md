# Architecture amended: drop asyncio, go fully sync to match SDK

## Rationale

SDK is sync-only (HarnessServiceClientSync, PcmRuntimeClientSync). No concurrent task execution needed. Asyncio.to_thread bridging adds complexity with zero benefit at 30 steps/task sequential. Decision: rewrite all async def → def, remove asyncio imports, match sample-agent structure exactly.
