# Gemini free tier: 5 req/min cap, retryDelay ~40s in error body

## Rationale

Free tier quota: GenerateRequestsPerMinutePerProjectPerModel = 5. Error includes retryDelay field (~40s). SDK default retry fires immediately and exhausts retries in <2s. Fixed: max_retries=0 on OpenAI client, custom retry loop in parse_step that extracts delay via regex 'retry in (\d+)s' and sleeps. Also applies exponential backoff for generic APIError.
