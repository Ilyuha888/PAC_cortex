# Tech stack: Python 3.12+, asyncio, openai SDK, httpx, pydantic

## Rationale

Chosen for async-first design with typed boundaries. openai SDK enables provider-agnostic LLM access via OPENAI_BASE_URL swap (pattern from sibling project muscat). httpx for BitGN API. pydantic-settings for env-based config. uv as package manager. Dev tools: pytest, pytest-asyncio, ruff. No frameworks — plain asyncio to keep control explicit.
