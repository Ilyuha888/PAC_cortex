# Sample agent uses SGR (NextStep Pydantic) — validates our deferred evaluation

## Rationale

The official BitGN sample agent uses `client.beta.chat.completions.parse(response_format=NextStep)` — exactly the SGR pattern we deferred. NextStep has: current_state (str), plan_remaining_steps_brief (List[str] 1-5 items), task_completed (bool), function (Union of all tool models). This means structured reflection IS the intended pattern. Our earlier deferral was correct to wait for task data, but now we have it — SGR is confirmed as the right core architecture. Budget is 30 steps (not 1000 API calls).
