# SGR evaluated for core agent loop — deferred until tasks are known

## Rationale

Schema-Guided Reasoning (abdullin.com/schema-guided-reasoning/) forces LLM through Pydantic-defined reasoning steps via constrained decoding. Strong fit for known-domain reasoning (compliance, forms) but poor fit for PAC's unknown task diversity. OpenAI function calling already provides constrained decoding at the action layer. SGR could add value at the meta-reasoning layer (plan/reflect before each tool call) but adds token overhead under budget pressure. Decision: defer until BitGN tasks are known. Revisit if tasks show repeatable reasoning patterns that could be schematized. Alternative considered: using SGR only for the reflect step, not full reasoning chain.
