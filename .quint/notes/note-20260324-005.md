# Agent loop: budget guard + stagnation detection

## Rationale

Core agent loop in solve_task() runs LLM chat with tool use until: (a) LLM returns text (final answer), (b) budget exhausted (1000 calls minus 50 reserve), or (c) stagnation detected (same tool call repeated 3 times). Budget is tracked per-run via BitgnClient._call_counts. Stagnation compares stringified call signatures.

- `src/pac_cortex/agent.py`
- `src/pac_cortex/client.py`
